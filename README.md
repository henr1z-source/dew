## Sobre a disciplina

Repositório de estudos da disciplina **DEW (Desenvolvimento Web)**. Aqui ficam anotações, exercícios e projetos desenvolvidos ao longo do curso, abrangendo desde a base de estruturação de páginas até o desenvolvimento com framework moderno.

---

## Tecnologias abordadas

### HTML5
Estrutura e semântica de páginas web. Tags, formulários, acessibilidade e boas práticas.

### CSS3
Estilização, Flexbox, Grid, animações, responsividade e variáveis CSS.

### JavaScript
Lógica de programação, DOM, eventos, fetch API, ES6+ e assincronismo.

### Angular
Framework SPA com TypeScript, componentes, serviços, rotas e data binding.
